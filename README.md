# App - Coin Converte
 Criando um aplicativo para conversão de moedas e câmbio.
 Aplicativo desenvolvido no Bootcamp Da DIO.
 
 Aplicativo desenvolvido em: 
 
 - MVVM
 - Coroutines
 - Retrofit
 - Koin
 - Room
